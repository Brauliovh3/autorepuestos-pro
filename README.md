# Repuestos Elva - Tienda Online de Repuestos Automotrices

## 🚗 Sobre el Proyecto

Repuestos Elva es una tienda online moderna y completa para la venta de repuestos automotrices en Perú. Desarrollada con HTML5, CSS3 y JavaScript vanilla, optimizada para despliegue en Vercel.

## ✨ Características Principales

### 🛍️ **Catálogo de Productos**
- Más de 50 productos en 6 categorías
- Búsqueda y filtrado avanzado
- Vista detallada de productos
- Sistema de calificación y especificaciones

### 🔐 **Sistema de Autenticación**
- Registro de usuarios con validación completa
- Inicio de sesión seguro
- Panel de usuario personalizado
- Historial de pedidos
- Almacenamiento local persistente

### 🛒 **Carrito de Compras**
- Gestión intuitiva del carrito
- Actualización en tiempo real
- Múltiples métodos de pago
- Checkout optimizado

### 🎨 **Diseño Profesional**
- Interfaz moderna y responsive
- Animaciones fluidas
- Optimizada para móviles
- Accesibilidad WCAG

### 📦 **Gestión de Pedidos**
- Sistema completo de pedidos
- Integración con usuarios
- Historial detallado
- Estados de pedido

## 🚀 **Despliegue en Vercel**

### Requisitos Previos
```bash
# Clonar el repositorio
git clone <repository-url>
cd pagina

# Instalar Vercel CLI (opcional)
npm i -g vercel
```

### Despliegue Automático
1. Conectar repositorio a Vercel
2. Configurar variables de entorno (si aplica)
3. Despliegue automático en cada push

### Despliegue Manual
```bash
# Usando Vercel CLI
vercel --prod

# O arrastrar la carpeta a vercel.com
```
